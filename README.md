# Retail Demand Forecasting & Inventory Optimization

## Project Overview

This project predicts future retail product demand using historical sales data from the M5 Forecasting Dataset. The objective is to help retailers optimize inventory, reduce stock shortages, and minimize overstock situations.

---

## Dataset

- M5 Forecasting Dataset
- Historical Walmart sales data
- Calendar information
- Product prices

---

## Project Workflow

1. Data Loading
2. Data Cleaning
3. Feature Engineering
4. Exploratory Data Analysis
5. Sales Forecasting using Random Forest Regressor
6. Model Evaluation
7. Save and Load Model

---
